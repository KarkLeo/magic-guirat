# Magic Guitar - User Stories

**Дата создания:** 2026-02-06
**Статус:** Planning

---

## Epic 1: Аудио Захват и Анализ

### US-1.1: Захват звука с микрофона
**Как** пользователь
**Я хочу** чтобы приложение захватывало звук с моего микрофона
**Чтобы** система могла анализировать мою игру на гитаре

**Критерии приемки:**
- [ ] Запрос доступа к микрофону
- [ ] Реализован аудио стрим в реальном времени
- [ ] Обработка ошибок доступа к микрофону
- [ ] Индикатор уровня входного сигнала

### US-1.2: Определение нот и струн
**Как** пользователь
**Я хочу** чтобы система определяла ноты которые я играю
**Чтобы** видеть какие струны я задействую

**Критерии приемки:**
- [ ] Частотный анализ аудио (FFT)
- [ ] Детекция питча (определение ноты)
- [ ] Маппинг нот на струны гитары (E, A, D, G, B, E)
- [ ] Обработка полифонии (несколько струн одновременно)

### US-1.3: Распознавание аккордов
**Как** пользователь
**Я хочу** чтобы система определяла аккорды
**Чтобы** видеть их названия на экране

**Критерии приемки:**
- [ ] База данных популярных аккордов
- [ ] Алгоритм распознавания комбинаций нот
- [ ] Определение типа аккорда (мажор, минор, септаккорды и т.д.)
- [ ] Отображение названия аккорда

## Epic 2: Визуализация

### US-2.1: Базовая визуализация струн
**Как** пользователь
**Я хочу** видеть визуальное представление струн
**Чтобы** понимать какие струны звучат

**Критерии приемки:**
- [ ] 6 визуальных струн на темном фоне
- [ ] Свечение активных струн
- [ ] Анимация при звучании
- [ ] Плавные переходы между состояниями

### US-2.2: Магическая визуализация аккордов
**Как** пользователь
**Я хочу** видеть красивую визуализацию аккордов
**Чтобы** наслаждаться магическим эффектом

**Критерии приемки:**
- [ ] Светящиеся линии соединяющие ноты аккорда
- [ ] Появление точек на позициях нот
- [ ] Движение частиц в такт музыки
- [ ] Фиолетово-синяя цветовая палитра
- [ ] Плавная анимация появления/исчезновения

### US-2.3: Система частиц
**Как** пользователь
**Я хочу** видеть эффекты частиц
**Чтобы** визуализация была более живой и магической

**Критерии приемки:**
- [ ] Particles engine (Canvas/WebGL/Three.js)
- [ ] Эмиссия частиц при ударе по струне
- [ ] Интерактивные частицы реагирующие на звук
- [ ] Эффекты trail и glow
- [ ] Оптимизация производительности

### US-2.4: Названия аккордов на экране
**Как** пользователь
**Я хочу** видеть название текущего аккорда
**Чтобы** знать что я играю

**Критерии приемки:**
- [ ] Отображение названия аккорда
- [ ] Красивая типографика
- [ ] Анимация появления названия
- [ ] Позиционирование в композиции

## Epic 3: UI/UX и Настройки

### US-3.1: Темная тема с магической атмосферой
**Как** пользователь
**Я хочу** видеть темный интерфейс с магическими эффектами
**Чтобы** погрузиться в атмосферу

**Критерии приемки:**
- [ ] Темный фон (градиенты)
- [ ] Фиолетово-синяя палитра
- [ ] Световые эффекты и bloom
- [ ] Атмосферные фоновые элементы

### US-3.2: Настройки приложения
**Как** пользователь
**Я хочу** настраивать параметры приложения
**Чтобы** адаптировать под свои нужды

**Критерии приемки:**
- [ ] Выбор устройства ввода (микрофон)
- [ ] Настройка чувствительности
- [ ] Настройка визуальных эффектов
- [ ] Сохранение настроек
